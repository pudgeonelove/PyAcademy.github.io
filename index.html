Конфигурация сервера (Переменные)

Любая система начинается с настроек. Переменные хранят параметры, которые будут использоваться во всем скрипте. Зададим базовые параметры ноды.

init_node.py 
node_name = "Gateway-Alpha"
max_connections = 5000
firewall_active = True

print("Инициализация узла:", node_name)
print("Лимит соединений:", max_connections)
> Инициализация узла: Gateway-Alpha
> Лимит соединений: 5000

Вычисления нагрузки

Рассчитаем пропускную способность канала. Скрипт должен заранее знать, хватит ли нам ширины канала для передачи бэкапов дата-центра.

bandwidth_test.py 
file_size_gb = 150
speed_mbps = 800

# Перевод ГБ в Мегабиты (1 ГБ = 8192 Мегабит)
total_megabits = file_size_gb * 8192
transfer_time_sec = total_megabits / speed_mbps

print("Время передачи (сек):", transfer_time_sec)
> Время передачи (сек): 1536.0

Маршрутизация (If/Else)

Управление доступом. Если IP-адрес входящего пакета не входит в белый список локальной сети, мы обязаны сбросить соединение.

firewall_rules.py 
incoming_ip = "192.168.1.105"
allowed_subnet = "192.168.1."

# Проверяем, начинается ли IP с разрешенной подсети
if incoming_ip.startswith(allowed_subnet):
    print("Соединение установлено (TCP 22).")
else:
    print("ВНИМАНИЕ: Пакет отброшен (DROP).")
> Соединение установлено (TCP 22).

Очереди задач (Lists)

Списки [] идеальны для хранения логов или очередей процессов. Мы создадим очередь демонов (фоновых служб), которые должны запуститься при старте сервера.

startup_services.py 
services = ["sshd", "nginx", "postgresql", "docker"]

print("Служб в очереди:", len(services))
print("Приоритетная служба (Индекс 0):", services[0])
print("Последняя служба:", services[-1]) # Отрицательный индекс
> Служб в очереди: 4
> Приоритетная служба (Индекс 0): sshd
> Последняя служба: docker

Автоматизация (For)

Вместо того чтобы вручную проверять каждый порт, мы поручим это циклу for. Он пройдет по всему массиву и выполнит команду сканирования.

port_scanner.py 
critical_ports = [22, 80, 443, 3306]

for port in critical_ports:
    print("Сканирование порта", port, "...", "ЗАКРЫТ")
    
print("Аудит безопасности завершен.")
> Сканирование порта 22 ... ЗАКРЫТ
> Сканирование порта 80 ... ЗАКРЫТ
> Сканирование порта 443 ... ЗАКРЫТ
> Сканирование порта 3306 ... ЗАКРЫТ
> Аудит безопасности завершен.

ping_util.py 
# Объявление функции
def ping_server(ip_address):
    print("Отправка ICMP пакета на", ip_address)
