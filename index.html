Что такое переменные?

Переменные — это контейнеры для хранения данных. При разработке ИИ в них часто хранят настройки модели: имя нейросети, версию или её статус.

config.py 
# Настройки нашей нейросети
bot_name = "NeuralX"
model_version = 4.0
is_active = True

print("Система:", bot_name)
print("Версия ядра:", model_version)
> Система: NeuralX
> Версия ядра: 4.0

Математика в коде

Компьютеры обожают цифры. Давай напишем скрипт, который рассчитает, сколько времени потребуется нейросети для генерации пачки изображений.

calc_time.py 
# Вводные данные
batch_size = 15 # количество картинок
time_per_image = 1.2 # секунд на одну генерацию

# Расчет
total_time = batch_size * time_per_image

print("Генерация займет:", total_time, "секунд.")
> Генерация займет: 18.0 секунд.

Ветвления логики (If/Else)

Программа должна принимать решения. Напишем фильтр: если пользователь вводит слишком длинный промпт (запрос) для генерации логотипа, ИИ выдаст ошибку.

prompt_check.py 
prompt = "Нарисуй крутой логотип для бренда одежды в стиле минимализм"
max_length = 40

# Функция len() считает количество символов
if len(prompt) > max_length:
    print("ОШИБКА: Запрос слишком длинный!")
else:
    print("Запрос принят к обработке.")
> ОШИБКА: Запрос слишком длинный!

Списки (Lists)

Когда данных много, их неудобно хранить в разных переменных. Списки позволяют хранить целые коллекции — например, очередь запросов на создание мемов.

queue.py 
# Массив (список) задач квадратных скобках
tasks = ["мем про физику", "кот-программист", "киберпанк город"]

print("Всего задач в очереди:", len(tasks))
# В программировании счет начинается с нуля!
print("Сейчас обрабатывается:", tasks[0])
> Всего задач в очереди: 3
> Сейчас обрабатывается: мем про физику

Циклы (For)

Главная сила кода — автоматизация рутины. Цикл for позволяет пройтись по всему списку и выполнить действие с каждым элементом. Давай протестируем сразу несколько ИИ-моделей.

benchmark.py 
models = ["ChatGPT", "Gemini", "Claude"]

# Цикл берет каждую модель из списка по очереди
for ai in models:
    print("Запуск тестов для модели:", ai)
    
print("Все тесты успешно завершены.")
> Запуск тестов для модели: ChatGPT
> Запуск тестов для модели: Gemini
> Запуск тестов для модели: Claude
> Все тесты успешно завершены.
